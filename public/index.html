App version: 1.0.1

purr starter is a simple front-end starter kit/boilerplate built with modern web technologies.

It features: SASS, WebPack, Browsersync, Eslint with airbnb config, Some usefull npm scripts...

This starter-kit has been built so that it is as maintainable as possible. Webpack tasks can be easily modified and the content of the site can be adapted according to your desires. You can add loaders to Webpack, the CSS framework you want or any JS framework.
The idea is to have a simple starter-kit with Webpack that allows to start projects quickly.


You're seeing the demo app. Keep in mind that everything inside the ./src folder is for demonstration purpose and can be deleted/modified as you want using the npm script: npm run remove-demo or yarn remove-demo.
